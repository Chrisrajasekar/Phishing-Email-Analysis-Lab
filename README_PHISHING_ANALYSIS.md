
# PHISHING EMAIL ANALYSIS PROJECT

![Header Analysis](/mnt/data/header_analysis.png)
![Redirect Chain](/mnt/data/redirect_chain.png)

This repository contains a safe, documented, and fully reproducible lab for analyzing phishing emails in a controlled environment.

## Features
- Virtualized lab using VirtualBox/Kali/Windows.
- `.eml` phishing email parsing.
- Header analysis, URL tracing, attachment sandboxing.
- IOC extraction and reporting.
